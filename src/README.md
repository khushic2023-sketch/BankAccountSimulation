# BankAccountSimulation

## Description
This is a Java-based banking system that demonstrates object-oriented programming principles.
It supports multiple account types, transaction management, and interest calculation.
The project showcases inheritance, encapsulation, method overriding, and transaction logging.

## Features
- Abstract 'BankAccount' class as base for all account types
- 'CheckingAccount' with optional overdraft support
- 'SavingsAccount' with interest application
- Deposit and withdrawal functionality
- Transaction history tracking
- Validation to prevent invalid operations

## Technologies
- Java 17+
- Git for version control
